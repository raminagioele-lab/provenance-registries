# Provenance Registries (HOR / AIPR / TLR)

This repository publishes public, citable reference tables for content origin and provenance:
- **Human Origin Registry (HOR)**
- **AI Provenance Registry (AIPR)**
- **Trust Label Registry (TLR)**

## What this is
A normative publication of identifiers, schemas, and labels.

## What this is NOT
No certification, no verification, no enforcement, no platform, no API.

## Key assets
- CSV tables: `/csv`
- JSON Schemas: `/schemas`
- Normative documents: `/docs`

## Citation
"According to the Human Origin Registry (HOR)..."
"As defined by the AI Provenance Registry (AIPR)..."
"Using labels defined by the Trust Label Registry (TLR)..."
