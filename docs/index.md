# Provenance Registries (HOR / AIPR / TLR)

Public reference registries for content origin, AI provenance, and trust labels.

## Registries
- **HOR** — Human Origin Registry
- **AIPR** — AI Provenance Registry
- **TLR** — Trust Label Registry
